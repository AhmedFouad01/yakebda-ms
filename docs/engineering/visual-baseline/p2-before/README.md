# P2 Visual Baseline — Before

This directory is the required visual contract for the P2 maintainability refactor.

## Required screens

- POS
- KDS
- Orders detail
- Menu
- Customers
- Users

## Required viewports

- 1366×768
- 1920×1080

Capture Light and Dark for each screen where both themes are supported.

## Naming

`<screen>--<theme>--<width>x<height>.png`

Examples:

- `pos--light--1920x1080.png`
- `orders-detail--dark--1366x768.png`

## Gate

No CSS duplicate may be removed until the baseline pack is captured and reviewed. The only intentional visual change allowed during P2 is the RTL drawer entering from the right.
